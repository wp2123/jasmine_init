Game Guides

You have 6 times to guess the 4 bits random number.
If the number you inputted has one member the same as answer with location and value, it will return 1 A.
If the number you inputted has one member the same as answer only with value, it will return 1 B.
Example: Answer is 1234, what you inputted is 1498,then return 1A1B.
